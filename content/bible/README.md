# Bible text files

KJV, WEB, and ASV texts are generated by the content pipeline (`content-tools/`).

Run `uv run python -m scripturejam_tools.ingest.bible_text` to produce
`KJV.yaml`, `WEB.yaml`, and `ASV.yaml` in this directory.

These files are large (~17 MB total) and are **baked into the Docker image** at
build time (see `app/server/Dockerfile`). They are not committed to the
repository — add them to `.gitignore` if you generate them locally.

Each file is a YAML mapping: `book → chapter → verse → text`.

```yaml
Matthew:
  1:
    1: "The book of the generation of Jesus Christ..."
    2: "Abraham begat Isaac..."
```
